# 🎈 Кунгурские аэронавты

Аркадная игра: воздушный шар летит над городами России, уворачиваясь от деревьев, туч и птиц. Со сменой времени суток, бонусами и таблицей лидеров.

🎮 **Играть:** https://a750ak-blip.github.io/kungur-balloon/

## Технологии
- HTML5 Canvas — отрисовка игры
- JavaScript — игровая логика, физика, столкновения
- Web Audio API — звук без файлов (синтез)
- Управление: мышь, тач, клавиатура

## Про проект
Это **вайб-кодинг проект** — собран с помощью ИИ-инструментов.
Моё участие: идея, концепция, тема родного Кунгура, дизайн, тестирование и доведение до играбельного состояния.
